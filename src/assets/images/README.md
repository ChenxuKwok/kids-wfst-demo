# Images Directory

Place your image files here:
- architecture.png
- results.jpg
- thumbnail.jpg
- logo.png

Supported formats: PNG, JPG, JPEG, WebP, SVG

Example usage in config.js:
```javascript
images: [
  { 
    id: "architecture", 
    file: "images/architecture.png", 
    alt: "System Architecture Diagram" 
  }
]
```
