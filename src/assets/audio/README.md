# Audio Directory

Place your audio files here:
- sample1.mp3
- sample2.wav
- speech_example.mp3

Supported formats: MP3, WAV, OGG

Example usage in config.js:
```javascript
samples: [
  { 
    name: "Speech Sample 1", 
    file: "audio/sample1.mp3", 
    description: "Example speech with dysfluencies" 
  }
]
```
